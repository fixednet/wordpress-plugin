== Description ==

This plugin is an interface between your WordPress website and the maintenance services provided by MyManaged.Site and sister brands.
Features include:
- Tracking of WordPress state, including plugin and theme versions, server specification and configuration
- Alerts when state changes
- Remote updates of plugins and themes
- Secure authenticated login

Support available
This plugin does not function standalone and requires an account with MyManaged.Site or a sister brand.
For any support questions please log into your account with your website maintenance agency and contact them.
Support is available 24/7 via task thread.

== Installation ==

The plugin can be installed through the standard WordPress plugin install process.
1. In the plugins section of your WordPress dashboard, search for MyManagedSite, install it and activate it.
Alternatively, upload the MyManagedSite plugin folder to your /wp-content/plugins/ directory, or go to your WordPress admin area and click on Plugins Â» Add New. Then activate the plugin through the 'Plugins' menu in WordPress.
2. Visit the 'MyManagedSite' menu item in your admin sidebar. You will be prompted for your Site ID, which you can obtain through your website maintenance agency.

== Frequently Asked Questions ==

Q. Will I be able to track support requests and updates within my WordPress dashboard.
A. Not at present. This plugin is simply a link between the Mymanaged.site interface and your WordPress site that allows us to authenticate and gather debug data.
Q. Is this plugin actively maintained.
A. Yes, new features are in frequent development.
Q. Can I have assistance installing the plugin?
A. Yes. Contact your maintenance agency for assistance.
